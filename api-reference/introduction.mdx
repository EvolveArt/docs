# Pragma Node API

Welcome to the Pragma Node API documentation. This API provides access to both on-chain and off-chain price data, as well as optimistic oracle functionality and real-time data streaming through WebSocket connections.

## Base URL

```
https://{environment}.pragma.build/node/v1
```

Where `{environment}` can be:
- `devnet` - for testing purposes
- `prod` - for production use

## Authentication

Most endpoints require authentication using an API key. Include your API key in the request headers:

```bash
Authorization: Bearer YOUR_API_KEY
```

## API Groups

Our API is organized into the following main categories:

- **Offchain** - Access to aggregated price data and real-time market information
- **Onchain** - Interact with on-chain price feeds and historical data
- **Optimistic Oracle** - Access to Pragma's optimistic oracle functionality
- **WebSocket** - Real-time data streaming capabilities
